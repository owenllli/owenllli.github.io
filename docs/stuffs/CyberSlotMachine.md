# 赛博老虎机

## 项目简介

赛博老虎机 (Cyber Slot Machine) 是一个简单的网页 demo，模拟赛博老虎机的体验。用户可以通过点击按钮随机抽取内容，体验不同的超能力和挑战。

## 功能说明

- **你可以……**：展示用户可以进行的操作。
- **而且你……**：展示用户的额外能力或选项。
- **但是你……**：展示可能遇到的挑战或限制。
- **你的超能力是……**：展示随机生成的超能力。
- **但是……**：展示可能的后果或限制。
